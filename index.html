Docker for front-end developers

Without Docker

  1. $ brew update
  2. $ brew install mysql
  3. ...
  4. $ \curl -sSL https://get.rvm.io | bash
  5. ...
  6. $ rails s

Frontend

  1. Node.js >= 16
  2. npm >= 8

Backend

  1. go1.18
  2. MySQL
  3. Redis
  4. Elasticsearch
  5. Capistrano
  6. Syslog
  7. Postgresql(comments)

Server setup

  1. Install GVM(https://github.com/moovweb/gvm)
  2. gvm install go1.18.13 --binary
  3. gvm use go1.18.13 --default
  4. In the project folder create a link to golang
  5. Install gb go get github.com/constabulary/gb/...
  6. Configure authorization git config --global url."git@git......
  7. install dependencies gb vendor restore
  8. Install, setup and start DB (link: ....)
  9. npm run build (npm run build:dev)
  10. npm start

With Docker

  1. $ docker-compose up api
  2. ...
  3. Listening localhost:8080

Advantage of using docker

  1. Consistency
  2. Portability
  3. Isolation
